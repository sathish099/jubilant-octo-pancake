# Signature Menswear Website (Astro Version)

This is a modern and sleek website for Signature Menswear, an online clothing store based in Chennai. This version of the site is built with Astro, a modern frontend framework, to enable UI/UX animations and provide a more interactive user experience.

## Technologies Used

*   **Astro:** For the core framework and component-based architecture.
*   **HTML5 & CSS3:** For the structure, content, and styling of the website.

## How to Deploy

This Astro website can be deployed easily and for free on various hosting platforms that support modern JavaScript frameworks. We recommend using [Netlify](https://www.netlify.com/) or [Vercel](https://vercel.com/) for their seamless integration with Astro.

### Deploying to Netlify or Vercel

1.  **Push to a Git Repository:**
    Make sure your project is on a Git provider like GitHub, GitLab, or Bitbucket.

2.  **Sign up for a Hosting Provider:**
    Create a free account on [Netlify](https://app.netlify.com/signup) or [Vercel](https://vercel.com/signup).

3.  **Create a New Site/Project:**
    - From your hosting provider's dashboard, choose to create a new site/project and import the repository for this project.
    - The hosting provider will automatically detect that you're deploying an Astro site.

4.  **Configure Build Settings:**
    - **Build Command:** `npm run build`
    - **Publish Directory:** `dist`
    - These are the default settings for Astro projects and should be automatically configured for you.

5.  **Deploy:**
    - Click the "**Deploy**" button. The hosting provider will build and deploy your site, providing you with a live URL.

That's it! Your Astro-powered website is now live.
